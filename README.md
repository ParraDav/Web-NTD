Trabajamos 3 estudiantes de forma colaborativa usando GitHub como eje central. La estrategia se basó en una división modular: mientras desarrollamos el index.html en conjunto, cada uno se encargó de una sección específica (.html independiente). A través de constantes pushes, logramos integrar todas las partes en un solo flujo dinámico, asegurando un control de versiones eficiente y un trabajo en equipo organizado.
